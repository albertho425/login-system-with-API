# Login system with API

* Login and logout
* Edit your profile
* Dashboard that displays the following
    * Last profile edit date
    * IP address
    * City
    * Country
    * Weather temperature in F and C units
    * Weather icon
    * Weather description
    * Exchange rate with USD to your country's currency based on IP
